Summary

Graphics Engine made from a given base during the Bachelors Degree in Computer Science for Games on Sheffield Hallam. The aim of the project was create a suitable render engine on the first semester and expand it the second one to be able to create a game. (PONER AQUI ENLACE AL JUEGO CUANDO ESTE).
